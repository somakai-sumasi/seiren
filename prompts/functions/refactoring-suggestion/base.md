# 提案手順

1. まずコードを分析し、設計上の問題点を特定する
2. 各問題に対して具体的な解消方法を提案する
3. 改善後のクラス設計をMermaidクラス図で示す
4. リファクタリングの実行手順を提示する

# 重要な注意事項

- 改善案は具体的なクラス名・メソッド名を含めること
- Mermaidクラス図は改善後の設計を表現すること
- 段階的に適用可能なリファクタリング手順を示すこと
- 過度な抽象化を避け、実用的な提案をすること
